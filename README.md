# webfont_SweiSpring
The split subsetted webfont of Swei Spring.

[狮尾四季春](https://github.com/max32002/swei-spring)的切片子集化。

字体版本号 1.068。

使用方法：
```CSS
@import url('https://cdn.jsdelivr.net/gh/WinterholdPrime/webfont_SweiSpring@main/SweiSpringSugarSC-M.css');
@import url('https://cdn.jsdelivr.net/gh/WinterholdPrime/webfont_SweiSpring@main/SweiSpringSugarSC-B.css');
```
